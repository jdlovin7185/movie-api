Objective

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Technical Requirements

Request URL HTTP Method Request body data format Response body data format
Return a list of ALL movies to the user /movies GET None A JSON object holding data about ALL movies available to the user
Return data (description, genre, director, and image URL) about a single movie by title to the user /movies/:Title GET none a JSON object hold data about a single title as such: {title: "I am Legend", description: "I Am Legend is a 2007 American post-apocalyptic action thriller film loosely based on the 1954 novel of the same name by Richard Matheson.", genre: "Sci-fi/Horror", director: "Francis Lawrence", https://irs.www.warnerbros.com/keyart-jpeg/movies/media/browser/i_am_legend_key_art.jpg"}
Return data about a genre (description) by name/title (e.g., "Thriller") /movies/genre/:Title GET None a JSON object holding data about a single movies genre as such: {title: "Prometheus", genre: "Sci-fi/Horror"},
Return data about a director (bio, birth year, death year) by name /movies/director/:Name GET None a JSON about holding data about a single director as such: {director: [];}
Allow new users to register /users POST
Allow users to update their user info(username) /users/:Username POST a JSON object holding data about the user info to update, structured like: {name: "New Username", email: newusername@email.com} a message saying the user info has been updated successfully.
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added) /users/:Username/Movies/:FavoriteMovies PUT a JSON object holding data about the user info to add, as such: {name: "user", movies: {, title: '', stars: [] }} A message indicating the movie was added
Allow users to remove a movie from their list of favorites(showing only a text that a movie has been removed) /users/:Username/Movies/:FavoriteMovies DELETE A message indicating the movie was deleted
Allow existing users to deregister (showing only a text that a user email has been removed) /users/:Username Delete None A message indicating the user was deleted
